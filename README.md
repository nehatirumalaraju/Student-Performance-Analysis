# Student-Performance-Analysis
Automated Student Performance Analysis using Python, Pandas, and Streamlit. Generates Excel reports and visualizes factors affecting exam scores.

📊 Project Overview
Analyzed factors affecting student academic performance using a dataset of 32+ attributes. Built data visualizations, correlation analysis, and an automated report generation system using Python.

🛠 Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn | Jupyter Notebook | Excel

📁 Project Structure
Student-Performance-Analysis/
├── http://Untitled5.ipynb                 # Main analysis notebook
├── http://StudentPerformanceFactors.csv   # Dataset with 32+ student factors
└── Student_Reports/                # Automated generated reports
    └── Automated_Student_Report.xlsx

🔍 Key Insights
- Performed EDA to identify top factors impacting exam scores
- Created visualizations for gender, parental education, study time vs performance
- Generated automated Excel reports for quick insights
